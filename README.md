## Description

Uruguayn is a website that recreates the experience of a Uruguayan children's channel from the 1990s. Inspired by the nostalgia of classic television, it features fictional programming of iconic cartoons, interactive sections, and a retro design with visual elements characteristic of the era.

The site includes:
- 🏠 **Home**: Main section with simulated "LIVE" broadcast.
- 📅 **Programming**: Detailed schedule of cartoons from Monday to Friday and Saturdays.
- 🎨 **Cartoons**: Gallery of some classic shows with "watch" buttons (currently placeholders).
- 🎮 **Games**: Section under construction. Coming soon!
- 📞 **Contact**: Simple form for messages. Coming soon!

## Technologies Used

- **HTML5** 🌐: Site structure.
- **CSS3** 🎨: Retro styles with animations and table design.
- **JavaScript** ⚙️: Basic navigation between sections.
- **Font Awesome** ⭐: Icons for navigation.
- **Images** 📸: Multimedia resources for immersive experience.

## How to Run

1. Clone or download the repository.
2. Open the `index.html` file in any modern web browser.
3. Navigate through the sections using the navigation bar buttons.

No server or additional dependencies required, as it is a static website at this time.
